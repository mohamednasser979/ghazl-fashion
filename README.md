# Ghazl Fashion Store

Full Stack E-commerce Website built using **React, Node.js, Express, and MongoDB**.

This project provides a complete online fashion store with product management, shopping cart, order system, and admin dashboard.

---

## Features

- Product catalog
- Product filters (size & color)
- Shopping cart system
- Checkout system
- Order management
- Admin dashboard
- Dark mode
- Responsive design
- Product image gallery

---

## Tech Stack

### Frontend
- React
- Bootstrap
- Axios
- Context API

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer (image upload)
